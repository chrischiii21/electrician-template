## 💡 Compression Recommendations

### For AVIF Files (hero-1 to hero-4)
- Already in AVIF format (modern, efficient)
- Check file size - should be under 200KB each
- If larger, re-encode with quality 75-85

### For PNG Files (logo, overlay)
- Use TinyPNG or similar tool
- Consider converting logo to SVG for scalability
- Target: Under 50KB for logo

### For JPG Files (content images)
- Use ImageOptim, TinyJPG, or Squoosh
- Target quality: 80-85
- Target size: Under 150KB each
- Consider converting to WebP or AVIF for better compression

### For External Unsplash Images
- Already optimized by Unsplash CDN
- No action needed
- Consider downloading and self-hosting for better control
