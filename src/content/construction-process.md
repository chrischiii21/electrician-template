# Construction Process

## 01 - Design & Engineering
- **Description**: Collaborate with architects and engineers to design complete electrical systems for new construction. Load calculations, panel sizing, and code compliance planning.
- **Details**: Electrical design, Load analysis, Panel sizing, Code review

## 02 - Foundation & Rough-In
- **Description**: Install conduit, cable trays, and underground utilities during foundation phase. Rough-in all wiring during framing for optimal placement and efficiency.
- **Details**: Underground utilities, Conduit installation, Cable routing, Panel placement

## 03 - Trim-Out & Connections
- **Description**: Install outlets, switches, fixtures, and equipment after walls are closed. Final connections and device installation with precision and attention to detail.
- **Details**: Device installation, Equipment wiring, Fixture connections, Final testing

## 04 - Inspection & Commissioning
- **Description**: Comprehensive system testing, code inspections, and full commissioning. Complete documentation and handoff to building owner.
- **Details**: System testing, Code inspection, Commissioning, Documentation
